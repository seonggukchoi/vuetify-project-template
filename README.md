# Vuetify Project Template

## Scripts

### Run Server as Watch Mode

```bash
npm run start
```

### Run Server on Server

```bash
npm run start:dev
npm run start:stage
npm run start:prod
```

### Build

```bash
npm run build
npm run build:dev
npm run build:stage
npm run build:prod
```

### Test

```bash
npm run test:unit
npm run test:e2e
```

### Styling Code

```bash
npm run lint
```
